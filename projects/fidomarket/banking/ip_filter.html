IP-фильтр

Просмотр IP-фильтров доступен только связанным физическим лицам в соответствии с ролевой моделью. Для пользователей, которые не имеют таких прав, на странице вместо контента отражается сообщение "У Вас не доступа к содержимому этой страницы".

Кнопка Удалить выделенные есть только у сотрудников с 1-й группой подписей. Кнопка (если она есть) становится
активной при хотя бы одном отмеченном чекбоксе. Пользователь не может выбрать все чекбоксы (т.е. удалить IP-фильтр). Если выбраны все чекбоксы, кроме одного и пользователь пытается выбрать последний, у него должна появиться ошибка "В списке должен остаться хотя бы один IP-адрес.

При нажатии на кнопку Удалить выделенные появляется окно для выбора ЭЦП, до нажатия оно скрыто. При нажатии кнопки Отменить, список связанных физических лиц остается без изменений.

При нажатии на кнопку "Обзор" на экран выводится окно проводника для поиска файла ЭЦП. После выбора нужного файла, его название появляется в текстовом поле и становится активной кнопка "Применить".
При нажатии на нее инициируется процесс наложения ЭЦП. После наложения ЭЦП появляется поле для ввода ОТП





При нажатии кнопки возможно 3 сценария: 1-ОТП корректный и введен в течение своего срока действия. в таком случае нажатие кнопки приводит выведению страницы с измененными параметрами.
2-ОТП введен с ошибками или после окончания срока действия. В таком случае нажатие кнопки приводит к выводу ошибки
