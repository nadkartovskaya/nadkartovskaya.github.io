Счет Лимит



Если выбран чекбокс счета, поля для ввода лимита становится доступным для редактирования. В списке выводятся счета в грн. с масками 2600, 2650, 2620 (нотариусов и адвокатов), 2604

По умолчанию лимиты для клиента не установлены.

При нажатии на кнопку, поля, доступные для редактирования, становятся активными и их содержимое может быть изменено.

При нажатии на кнопку Сохранить появляется окно для выбора ЭЦП, до нажатия оно скрыто. При нажатии кнопки Отменить, профиль связанного физического лица остается без изменений.

При нажатии на кнопку "Обзор" на экран выводится окно проводника для поиска файла ЭЦП. После выбора нужного файла, его название появляется в текстовом поле и становится активной кнопка "Применить".
При нажатии на нее инициируется процесс наложения ЭЦП. После наложения ЭЦП появляется поле для ввода ОТП





При нажатии кнопки возможно 3 сценария: 1-ОТП корректный и введен в течение своего срока действия. в таком случае нажатие кнопки приводит выведению страницы с измененными параметрами.
2-ОТП введен с ошибками или после окончания срока действия. В таком случае нажатие кнопки приводит к выводу ошибки
