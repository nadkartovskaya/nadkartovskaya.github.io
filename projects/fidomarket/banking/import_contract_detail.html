Для внесения изменений в профиль контракта и/или для загрузки файлов контракта, пользователь должен нажать кнопку Редактировать. При нажатии кнопки Редактировать (в тех статусах, при которых кнопка активна), становятся доступными кнопки Удалить в таблице файлов контракта, становятся доступными для редактирования поля контракта, появляется блок: загрузка файлов контракта и кнопки Сохранить, Отменить, Отправить на верификацию. Для удобаства этот блок на странице помечен, как Блок редактирования. Во всех других случаях этот блок скрыт. Т. е. пользователь имеет возможности внести необходимые изменения, сохранить их и отправить на верификацию.

Блок редактирования





При нажатии на кнопку Обзор, открывается окно проводника и пользователь получает возможность выбрать файлы контракта. При нажатии на кнопку Загрузить, инициируется процесс загрузки файлов контракта Допускается загрузка нескольких файлов. Для контрактов в статусах Новый, Отклонен, На верификации допускается удаление файлов и повторная загрузка.

Уточнить допустимые форматы файлов и размеры. Сделать подсказку по этому поводу.

После нажатия кнопки профиль контракта сохраняется, пользователь остается на той же странице. Кнопка Отменить становится неактивной и пользователь может отправить контракт на верификацию.
