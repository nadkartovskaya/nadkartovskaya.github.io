Поиск клиента







Одно из полей должно быть заполнено согласно требований: Название - текст, ЭДРПОУ - число из 8-ми символов, ИНН - число из 12 символов. Можно объединить все поля в одно и искать по любому из признаков.


В результат нажатия кнопки выводится список клиентов, параметры которых удовлетворяют заполненным параметрам поиска
